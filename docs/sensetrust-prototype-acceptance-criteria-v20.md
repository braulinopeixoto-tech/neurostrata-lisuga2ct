# SenseTrust Prototype Acceptance Criteria v2.0

Aceite:
- usuario entende o que a SenseTrust certifica;
- usuario entende o que nao certifica;
- usuario entende que nao ha coleta real;
- usuario entende que e prototipo;
- CTA nao induz promessa indevida;
- verificacao publica nao expoe dado clinico;
- rotas sao navegaveis;
- disclosure e visivel;
- nenhuma claim proibida aparece como promessa ativa.

Referencias obrigatorias: Website Blueprint v1.9, Public Narrative v1.8 e Investor Room v1.7.
