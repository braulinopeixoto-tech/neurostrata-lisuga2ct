# Manual Supabase Apply Instructions - SenseTrust

## Status

Fallback manual necessario porque `npx` e `supabase` CLI nao estao disponiveis neste ambiente local.

## Acao unica no Supabase SQL Editor

1. Abrir o projeto Supabase `yponblaeampkodzjrjko`.
2. Abrir SQL Editor.
3. Abrir `docs/manual-supabase-apply-sensetrust.sql` neste repositorio.
4. Pressionar Ctrl+A no conteudo do arquivo SQL.
5. Colar todo o conteudo no SQL Editor.
6. Clicar em Run.
7. Depois rodar as queries de validacao em `docs/supabase-validation-queries.sql`.

## Regra

Nao usar o caminho do arquivo como SQL. O SQL Editor deve receber o conteudo completo do arquivo `manual-supabase-apply-sensetrust.sql`.
