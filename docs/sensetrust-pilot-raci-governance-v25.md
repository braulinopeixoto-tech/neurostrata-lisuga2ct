# SenseTrust Pilot RACI Governance v2.5

RACI organiza responsavel, aprovador, consultado e informado para cada piloto institucional simulado.

As revisoes obrigatorias sao:

- Humana.
- Juridica.
- Institucional.
- Tecnica.
- Etica.

RACI nao cria contrato, assinatura legal, cliente real ou parceria formalizada. Ele apenas documenta responsabilidade simulada e governanca de execucao.
