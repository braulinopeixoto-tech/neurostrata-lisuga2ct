# SenseTrust SEO Safe Language v1.9

Status: REQUIRED_REVIEW

Linguagem segura:
- confianca documental;
- integridade documental;
- verificacao publica segura;
- metadata_only;
- rastreabilidade e versionamento;
- governanca de documentos.

Evitar:
- promessa de diagnostico correto;
- promessa de resultado clinico;
- alegacao de validade juridica plena;
- alegacao de producao real;
- alegacao de receita, billing ou clientes reais.

SEO deve reforcar autoridade institucional sem capturar linguagem medica promocional.
