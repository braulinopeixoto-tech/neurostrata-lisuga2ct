# SenseTrust Pilot Checkpoints v2.5

Checkpoints obrigatorios:

- Checkpoint de escopo.
- Checkpoint de material.
- Checkpoint de demo.
- Checkpoint de feedback.
- Checkpoint de risco.
- Checkpoint de aceite.
- Checkpoint de interrupcao.
- Checkpoint executivo.

Cada checkpoint deve registrar status, criterios, bloqueios e evidencias exigidas. Falha de checkpoint pode pausar, revisar ou bloquear o piloto simulado.
