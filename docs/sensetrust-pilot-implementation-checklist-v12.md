# SenseTrust Pilot Implementation Checklist v1.2

## Pre-piloto

- [ ] participante selecionado;
- [ ] responsavel definido;
- [ ] termo de uso enviado;
- [ ] consentimento enviado;
- [ ] contrato-piloto validado;
- [ ] restricoes compreendidas;
- [ ] dados simulados preparados.

## Implantacao

- [ ] organizacao simulada criada;
- [ ] usuario criado;
- [ ] plano simulado vinculado;
- [ ] demo carregada;
- [ ] portal publico testado;
- [ ] audit report testado;
- [ ] Obsidian/MOC apresentado.

## Execucao

- [ ] fluxo end-to-end rodado;
- [ ] feedback coletado;
- [ ] riscos registrados;
- [ ] duvidas respondidas.

## Fechamento

- [ ] relatorio final;
- [ ] aceite ou nao aceite;
- [ ] decisao go/no-go;
- [ ] proximos passos.
