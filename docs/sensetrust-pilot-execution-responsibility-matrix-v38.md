# SenseTrust Pilot Execution Responsibility Matrix v3.8

## Objetivo
Distribuir papéis simulados de governança para uma execução não operacional, posterior à v3.7, vinculada à v3.6 e Lean v1.2.

## Critérios
Responsabilidades apoiam Go/Pause/Refine/Stop, evidências mínimas, decisão humana obrigatória e rastreabilidade BLC.

## Limites
Não cria contrato, cliente ou parceria. Trust Layer, NeuroStrata, VitalStrata, DNDA, metadata_only, LGPD, neurodireitos, simulated_only, no real pilot, no clinical data, no patient data, no diagnostic truth certification. DNDA significa Diagnóstico Neurofuncional Dimensional Auditável.
