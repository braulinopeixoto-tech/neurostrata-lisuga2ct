# SenseTrust SaaS Pilot Readiness

Status: not ready for real pilot, simulated readiness model only.

## Validated in v0.9

- simulated organizations;
- simulated users;
- role and permission matrix;
- commercial plan model;
- usage ledger;
- plan limit enforcement;
- simulated plan upgrade;
- tenant isolation contract;
- public portal metadata-only boundary.

## Blockers

- real billing is not implemented;
- payment gateway is not integrated;
- advanced authentication is not implemented;
- production Supabase RLS must be reconciled;
- legal and privacy review required;
- no real clinical data allowed.

## Closed pilot preparation

The SaaS Core can support a closed pilot specification after production-grade auth, billing decision, RLS, support process and LGPD review are completed.
