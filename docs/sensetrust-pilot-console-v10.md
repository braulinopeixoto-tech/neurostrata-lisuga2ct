# SenseTrust Pilot Console v1.0

Status: implemented locally, pending final build, Obsidian write and Git remote confirmation.

## Objective

The Pilot Console is a simulated operational demo that connects SenseTrust v0.3 through v0.9 in one end-to-end flow.

## What it connects

- organization, user and plan from SaaS Core v0.9;
- public verification portal from v0.8;
- signature, timestamp and emission integrity from v0.7;
- document state from v0.6;
- clinical commit chain from v0.5;
- Evidence Manifest and DNDA Trust Object from v0.4;
- public certificate and QR concept from v0.3.

## Boundary

This is not clinical production. It uses simulated metadata only and does not include real patient, CPF, EEG, qEEG, report, anamnesis, medication or clinical data.

## Public language

SenseTrust certifies integrity, provenance, document state and verifiability. It does not certify absolute diagnostic truth.
