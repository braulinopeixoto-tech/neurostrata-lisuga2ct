# SenseTrust Demo Governance v2.1

Governanca:
- quem pode demonstrar: apresentador treinado;
- para quem pode demonstrar: publico definido e controlado;
- materiais permitidos: prototipo local, docs v2.1 e disclosures;
- materiais proibidos: dados reais, contratos reais, claims absolutos;
- gravacao: apenas com aprovacao previa;
- reuniao publica: exige revisao institucional;
- reuniao institucional: permitida com roteiro;
- investidor: sem receita real;
- juridico: sem validade plena declarada;
- saude publica: sem Gov.br, ICP ou uso assistencial real.

Toda demo exige aprovacao humana quando houver publico externo.
