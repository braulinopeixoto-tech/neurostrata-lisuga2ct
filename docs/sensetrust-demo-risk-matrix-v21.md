# SenseTrust Demo Risk Matrix v2.1

Riscos:
- entender prototipo como producao;
- entender formulario como coleta real;
- entender QR como diagnostico;
- entender assinatura como legal real;
- entender receita como real;
- entender integracao Gov.br/ICP como ativa;

- extrapolacao clinica;
- uso indevido em proposta publica.
