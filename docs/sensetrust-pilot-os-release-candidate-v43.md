# SenseTrust Pilot OS Release Candidate v4.3

Objetivo: consolidar v3.7, v3.8, v3.9, v4.0, v4.1 e v4.2 em um Release Candidate simulado, demonstrável, auditável e supervisionado, com Lean v1.2 como trilha de congelamento.

A SenseTrust v4.3 não executa operação real, não formaliza parceria e não certifica resultado clínico, científico, regulatório ou comercial. Ela consolida um Release Candidate simulado, auditável e supervisionado do SenseTrust Pilot OS, reunindo navegação, critérios de aceite, trilha probatória, guardrails, prontidão de demonstração e preparação institucional para revisão humana.

Guardrails: simulated_only, metadata_only, human_review_required, release_candidate_only, demo_preparation_only, command_center_linked, LGPD, neurodireitos, DNDA = Diagnóstico Neurofuncional Dimensional Auditável. Regra terminológica permanente respeitada. Preparação v4.4 apenas simulada.
