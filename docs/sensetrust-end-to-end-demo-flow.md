# SenseTrust End-to-End Demo Flow

The v1.0 flow performs a simulated demo sequence:

1. organization selected;
2. user authorized;
3. plan validated;
4. simulated document created;
5. certificate created;
6. Evidence Manifest created;
7. DNDA Trust Object created;
8. Clinical Commit Chain created;
9. document state applied;
10. simulated signature applied;
11. logical timestamp applied;
12. emission hash calculated;
13. public portal generated;
14. usage ledger updated;
15. audit report generated.

All steps are metadata-only and simulated.
