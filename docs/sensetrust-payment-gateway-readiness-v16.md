# SenseTrust Payment Gateway Readiness v1.6

## Requisitos para gateway futuro

LGPD, minimizacao, logs auditaveis, ambiente sandbox, revisao juridica, revisao fiscal, revisao contabil e segregacao de dados.

## Vendor selection

Comparar fornecedores apenas como avaliacao futura. Nenhum gateway real esta implementado.

## Dados que nao podem ser enviados

Paciente real, dado clinico, CPF, exame, laudo, anamnese, medicamento, CID ou conteudo assistencial.

## Condicao de bloqueio

Bloquear se houver tentativa de gateway real sem politica de dados, sem contrato final, sem revisao fiscal ou sem governanca de privacidade.
