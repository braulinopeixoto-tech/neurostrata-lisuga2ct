# SenseTrust Pilot Presentation Script

## Opening

"A SenseTrust e a camada de confianca documental da NeuroStrata/DNDA/VitalStrata. Ela nao substitui julgamento clinico e nao certifica verdade diagnostica absoluta. Ela certifica integridade, proveniencia, estado documental, emissao e verificacao publica segura."

## Demonstration

1. Selecionar organizacao simulada.
2. Selecionar usuario com papel autorizado.
3. Conferir plano e limite.
4. Rodar fluxo end-to-end.
5. Ver cadeia de etapas.
6. Abrir certificado.
7. Ver Evidence Manifest.
8. Ver DNDA Trust Object.
9. Ver Clinical Commit Chain.
10. Ver assinatura e timestamp.
11. Ver estado documental.
12. Abrir portal publico.
13. Mostrar que dado clinico nao aparece.
14. Mostrar usage ledger.
15. Mostrar audit report.

## Closing

"Esta versao esta pronta para piloto fechado com dados simulados ou ambiente controlado. O proximo passo e escolher 3 a 5 profissionais/organizacoes para teste supervisionado."
