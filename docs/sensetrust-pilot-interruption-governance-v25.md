# SenseTrust Pilot Interruption Governance v2.5

Interruption Governance define quando pausar, bloquear, revisar escopo, acionar juridico, acionar direcao, arquivar ou encerrar piloto simulado.

Gatilhos:

- Pedido de dado clinico real.
- Ambiguidade contratual.
- Nome institucional sem autorizacao.
- Promessa diagnostica.
- Tentativa de operar clinicamente.
- Tentativa de declarar cliente ou parceria formalizada.

Toda decisao deve ser registrada em decision log metadata_only.
