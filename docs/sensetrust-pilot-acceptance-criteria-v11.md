# SenseTrust Pilot Acceptance Criteria v1.1

- organizacao criada;
- usuario autorizado;
- plano vinculado;
- fluxo end-to-end concluido;
- QR/portal publico acessivel;
- nenhum dado clinico no portal;
- audit report gerado;
- usage ledger atualizado;
- MOCs navegaveis;
- compreensao do usuario validada;
- feedback coletado;
- riscos mapeados;
- decisao go/no-go registrada.

Aceite minimo: o participante deve compreender a diferenca entre integridade documental e verdade diagnostica absoluta.
