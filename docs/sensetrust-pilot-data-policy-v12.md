# SenseTrust Pilot Data Policy v1.2

## Dados permitidos

- nome institucional;
- e-mail institucional;
- papel no piloto;
- feedback operacional;
- metadados simulados;
- usage ledger simulado;
- audit report metadata_only.

## Dados proibidos

- paciente real;
- CPF;
- dado clinico identificavel;
- EEG real;
- qEEG real;
- laudo real;
- anamnese real;
- medicacao real;
- escalas reais;
- documentos sensiveis.

## Dados simulados

O piloto deve usar dados simulados ou objetos documentais controlados sem pessoa identificavel.

## Retencao e exclusao

Dados institucionais minimos e feedback podem ser retidos pelo periodo do piloto e removidos mediante encerramento, salvo necessidade de auditoria documental metadata_only.

## Logs

Logs devem ser metadata_only e nao podem conter conteudo clinico.

## Principios

LGPD, neurodireitos, minimo necessario, rastreabilidade e vedacao de dado clinico sensivel.
