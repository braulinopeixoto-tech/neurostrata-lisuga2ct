# SenseTrust Execution Risk Register v2.5

Riscos controlados:

- Piloto virar operacao real.
- Uso de dados clinicos reais.
- Uso de nome institucional sem autorizacao.
- Interpretacao como cliente formal.
- Interpretacao como contrato.
- Interpretacao como validacao juridica plena.
- Promessa diagnostica.
- Material nao autorizado.
- Falha de revisao humana.
- Ausencia de regra de interrupcao.

Cada risco deve ter impacto, mitigacao e regra de interrupcao associada.
