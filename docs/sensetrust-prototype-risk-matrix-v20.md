# SenseTrust Prototype Risk Matrix v2.0

Riscos:
- usuario entender como site publicado;
- usuario entender formulario como coleta real;
- usuario entender verificacao como diagnostico;
- investidor interpretar receita como real;
- parceiro interpretar assinatura como legal real;
- instituicao interpretar Gov.br ou ICP-Brasil como ativo;
- ausencia de disclosure em rota sensivel.

Mitigacoes:
- disclosure global;
- CTAs simulados;
- envio bloqueado;
- copy metadata_only;
- revisao juridica, regulatoria, clinica e institucional antes de qualquer publicacao.
