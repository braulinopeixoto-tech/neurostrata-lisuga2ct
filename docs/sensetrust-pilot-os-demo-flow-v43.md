# Pilot OS Demo Flow v4.3

Objetivo: preparar fluxos de demonstração para governança interna, BioStrata Agro, investidor e jurídico, usando a trilha v3.7 a v4.2 e preparando v4.4 apenas como distribuição simulada.

Demo flow não é envio real, não chama API, não usa CRM, não envia email e não coleta dado clínico, paciente, amostra biológica, EEG/qEEG ou biomarcador real.

Guardrails: simulated_only, metadata_only, human_review_required, release_candidate_only, demo_preparation_only, command_center_linked, LGPD, neurodireitos, DNDA = Diagnóstico Neurofuncional Dimensional Auditável.
